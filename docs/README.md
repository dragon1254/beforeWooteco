### 처리 순서

1. 구입 금액 입력 받음

2. 구매 장수와 그 내역 처리 하여 출력

3. 당첨번호 입력 받음

4. 보너스 번호 입력 받음

5. 당첨번호와 보너스에 일치시켜서 당첨 여부 확인 처리

6. 당첨 통계 출력

### 입력사항

1. 구입 금액 입력

2. 당첨번호 입력

3. 보너스 번소 입력
